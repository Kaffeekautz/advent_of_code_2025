# Advent of Code 2025

## Struktur

- `day01/`, `day02/`, ...: Lösungen für die jeweiligen Tage
- `requirements.txt`: Python-Abhängigkeiten
- `.venv/`: Virtuelle Umgebung (nicht im Repo enthalten)

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Nutzung

Jeder Tag hat einen eigenen Ordner mit einer Lösung.
